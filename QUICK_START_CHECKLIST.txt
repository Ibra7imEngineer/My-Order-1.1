📋 IMMEDIATE NEXT STEPS:

[ ] 1. Get Your Gemini API Key
     └─ Visit: https://aistudio.google.com/app/apikey
     └─ Click "Get API key"
     └─ Copy the key

[ ] 2. Set API Key (Choose ONE):
     
     Option A - .env.php (Windows/Any):
     ├─ Copy: .env.php.template → .env.php
     ├─ Edit .env.php
     └─ Replace 'YOUR_GEMINI_API_KEY_HERE' with your key
     
     Option B - Environment Variable (Windows - PowerShell as Admin):
     ├─ Run: [Environment]::SetEnvironmentVariable("GEMINI_API_KEY", "YOUR_KEY", "User")
     └─ Restart Apache & PowerShell
     
     Option C - Environment Variable (Linux/Mac):
     ├─ Run: export GEMINI_API_KEY="YOUR_KEY"
     └─ Add to ~/.bashrc or ~/.bash_profile for persistence

[ ] 3. Restart Apache
     ├─ Open XAMPP Control Panel
     ├─ Click "Stop" on Apache
     ├─ Wait 2 seconds
     └─ Click "Start" on Apache

[ ] 4. Test Your Setup
     ├─ Open: http://localhost/My-Order/
     ├─ Click chat icon (floating button)
     ├─ Type a message
     ├─ Press Enter
     └─ Should see AI response

[ ] 5. Verify with Developer Tools
     ├─ Press F12 to open DevTools
     ├─ Go to Network tab
     ├─ Send another chat message
     ├─ Look for POST request to: /api/chat_handler.php
     ├─ Click it and verify:
     │  ✅ Status: 200
     │  ✅ No API key in "Request" tab
     │  ✅ Has text response in "Response" tab
     └─ If you see error, check the Response tab for details
